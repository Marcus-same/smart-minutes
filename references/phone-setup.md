# 手机物理按键配置：一键启动飞书妙记录音

## iPhone 方案

### 方案 A：Action Button（iPhone 15 Pro / 16 Pro 系列，推荐）

这是唯一真正的"物理按键一键启动"方案：

1. 打开「设置」→「操作按钮」
2. 左右滑动选择「控制」
3. 点击下方箭头展开控制列表
4. 找到「实时转录」（飞书图标），点击选中
5. 返回

**使用**：开会时长按左侧 Action Button（静音键位置），飞书自动开始实时转录录音。

**要求**：飞书 iOS 7.63+

### 方案 B：锁屏 Widget（iPhone 支持 iOS 16+）

1. 在锁屏界面长按 → 点击「自定义」
2. 选择「锁屏屏幕」→ 点击下方 Widget 区域
3. 搜索「飞书」→ 选择「实时转录」
4. 拖动排列位置 → 完成

**使用**：锁屏状态下点击 Widget 即可跳转飞书实时转录。

**要求**：飞书 iOS 7.63+，iOS 16+

### 方案 C：主屏幕 Widget

1. 长按主屏幕空白处 → 左上角「+」
2. 搜索「飞书」→ 选择 Widget 样式
3. 滑动到「实时转录」变体 → 添加

**优缺点**：需要解锁手机再按，比 Action Button 多一步。

### 方案 D：Back Tap（背面轻点，iOS 辅助功能）

1. 「设置」→「辅助功能」→「触控」→「背面轻点」
2. 选择「轻点两下」或「轻点三下」
3. 在快捷指令列表中…**注意：飞书实时转录不是快捷指令操作，此方案不可用。**

背面轻点只能触发系统级快捷指令，无法直接启动飞书妙记。不推荐。

## Android 方案

Android 没有 Action Button 那样的物理按键捷径，但有几种方式做到"最短路径启动录音"。

### 方案 A：桌面 Widget（推荐，所有 Android 通用）

1. 长按桌面空白处 →「添加工具」/「Widget」
2. 搜索「飞书」→ 找到「实时转录」Widget
3. 拖到桌面

**使用**：解锁后点一下 Widget → 飞书直接打开实时转录页。

**要求**：飞书 Android 7.67+

### 方案 B：锁屏 Widget（部分机型支持）

三星（One UI）、小米（MIUI）、OPPO（ColorOS）等在锁屏界面左滑可添加 Widget：

1. 锁屏界面 → 左滑进入 Widget 面板
2. 添加「飞书-实时转录」
3. 锁屏状态直接点击

**注意**：原生 Android / Pixel / 一加等可能不支持锁屏 Widget，只能用桌面方案。

### 方案 C：侧键绑定（三星 / 小米 / OPPO 等）

虽然不能直接跳到录音页，但可以一键打开飞书，然后手动点实时转录入口：

**三星**：设置 → 高级功能 → 侧键 → 按两下 → 打开应用 → 飞书

**小米**：设置 → 更多设置 → 快捷手势 → 按键快捷方式 → 飞书

**OPPO/一加**：设置 → 便捷工具 → 手势体感 → 息屏手势 / 按键快捷方式

**华为**：设置 → 辅助功能 → 快捷启动及手势 → 快捷启动 → 飞书

约束：侧键只能启动飞书 App，不能直达录音页。仍需在 App 内再点一次实时转录。

**如果飞书支持 App Shortcut（长按图标弹出菜单）且其中有「实时转录」，长按飞书图标点一下就进录音页了。** 目前飞书 Android 版是否带这个快捷方式需实测确认——若有，链路就是：长按飞书图标 → 点实时转录 = 2 步。

### 方案 D：快捷指令 / 自动化（Tasker / MacroDroid）

飞书没有提供 URL scheme 来启动录音，所以自动化工具也无法绕过 App 内部导航。不推荐。

### 各方案对比

| 方案 | 操作步数 | 通用性 | 能否物理按键触发 |
|------|---------|--------|---------------|
| 桌面 Widget | 解锁 + 1 步 | 全机型 | 否 |
| 锁屏 Widget | 1 步 | 三星/小米/OPPO | 否 |
| 侧键绑飞书 | 2 步（按侧键+点录音） | 主流品牌 | 近物理，但多一步 |
| App 长按快捷 | 1 步 | 取决于飞书版本 | 否 |

**最佳体验**：桌面 Widget（一触即达）+ 侧键绑飞书（备用，物理按键接近感）

## 会后分享流程

录音结束后，把妙记链接发给 Claude Code：

**方法 1 — 飞书 Bridge 群**：
1. 打开飞书 →「云空间」→「妙记」
2. 找到刚录的妙记 → 右上角「...」→「分享」→「复制链接」
3. 粘贴到「Claude Code Bridge」群
4. Claude Code 轮询群消息读到后自动处理

**方法 2 — 直接对话**：
1. 复制妙记链接
2. 在 Claude Code 对话中粘贴，说"帮我生成智能纪要"或直接说"这个妙记 kuakua 一下"

## 常见问题

**Q: 为什么设置里找不到"实时转录"控件？**
A: 检查飞书版本是否 ≥ iOS 7.63 / Android 7.67。更新飞书后重启手机。

**Q: Action Button 长按后没反应？**
A: 确认飞书已登录且有网络。首次使用需在飞书中授权麦克风权限。

**Q: 录音能录多久？**
A: 飞书妙记最长 6 小时，单文件最大 6GB。手机端受电量和存储影响。

**Q: 录音中断了怎么办？**
A: 飞书会自动保存已录部分。在妙记列表中可以找到未完成的录制，仍然可以转写。
